# 222_64 章节编号自定义（支持汉字、字母、罗马数字）

## 如何测试
- 测试文档：TeXmacs/tests/tmu/222_64.tmu
    1. 清理缓存
    2. 插入一个章环境，点击小扳手，看是否有"Number style"分组和7个选项
    3. 测试选择不同的编号样式，章节标题应相应变化
    4. 测试目录中的章节编号是否也随之变化
    5. 测试节、子节、小节是否也可以修改
    6. 测试不同语言环境（英语、中文等）下是否正常工作

## 2026/03/26
### What
在章节标题的首选项（小扳手）中添加编号样式选项，支持：
- 阿拉伯数字 (1, 2, 3) - 默认
- 汉字数字 (一, 二, 三)
- 大写罗马数字 (I, II, III)
- 小写罗马数字 (i, ii, iii)
- 大写字母 (A, B, C)
- 小写字母 (a, b, c)
- 圆圈数字 (①, ②, ③)

补充范围：
- 编号样式可配置范围扩展到 `chapter`、`section`、`subsection`、`subsubsection`
- 不提供编号样式切换 `paragraph`、`subparagraph`
- 不适用标签不显示可配置项 `chapter*`、`section*`、`subsection*`、`subsubsection*`

### Why
原先章节编号只能通过修改宏来更改格式，对普通用户不友好，因此提供一个直观的首选项选项来切换编号样式。

### How
修改下列文件
1. `TeXmacs/packages/section/section-base.ts` - 添加 `chapter-number-style` 变量和 `the-chapter` 宏
2. `TeXmacs/progs/text/text-menu.scm` - 在小扳手菜单中增加编号样式入口，并按标签类型控制显示范围
