# [201_73] 点击序言菜单后自动添加空行

## 如何测试
1. 新建文档，样式选择 book以及generic
2. 点击 `插入` -> `章节`
3. 选择 `序言`
4. 检查序言下方是否自动有一个空行

## 2026/2/6
### What
点击"序言"菜单后，在章节下方自动插入一个空行

### How
修改 `TeXmacs/progs/text/text-menu.scm` 第138行

```scheme
("Prologue::menu"
  (begin
    (make-unnamed-section 'prologue)
    (insert-return)))
```
